Quiz application created in PHP with PDO interface and MySQL databases. 
Test your programming knowledge. 

DATABASE: 
In file "database" you can see database "phpquizz.sql". 
Use your localhost/phpMyAdmin and import this database to your server.

Also in file "database" you can open "database.php" on your local server, 
just type in: localhost/restfulapi/database/database.php and database 
will be created on your phpMyAdmin.
